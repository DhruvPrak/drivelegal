# DriveLegal 🚦

AI-powered Indian traffic law platform built for the National Road Safety Hackathon 2026 (CoERS, RBG Labs, IIT Madras).

## Features
- GPS-based traffic law lookup
- Challan calculator for all states
- RTO office directory
- AI plain-language explainer
- Know Your Rights guide
- Multilingual (English, Hindi, Tamil)
- Challan history tracker
- AI chat assistant (RAG)
- PWA offline support
- Law change alerts
- WhatsApp sharing

## Tech Stack
- Frontend: React 18 + Vite + Tailwind CSS
- Backend: FastAPI + PostgreSQL + PostGIS + pgvector
- AI: Anthropic Claude API
- Cache: Redis
- Deployment: Vercel + Railway.app
